Lista 1 - Questão 3

Qual é a diferença entre linguagem de programação Server Side e Client Side? Cite exemplos de cada tipo.

As linguagens de programação Server Side e Client Side são utilizadas para criar aplicações web, mas têm funções e execuções em momentos distintos.

Server Side: São linguagens que executam no servidor. O código é processado no servidor antes de ser enviado ao cliente . Isso significa que o cliente não tem acesso direto ao código-fonte. Essas linguagens são usadas para processar dados, interagir com bancos de dados e gerar o conteúdo dinâmico da página.

Exemplo de Server Side: PHP, Python (com frameworks como Django ou Flask), Ruby on Rails, Node.js .

Client Side: São linguagens que executam no navegador do cliente. O código é enviado para o cliente em formato de arquivos HTML, CSS e JavaScript, que são interpretados pelo navegador. Essas linguagens são usadas para criar interatividade nas páginas, como animações, validação
